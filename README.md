# 🔧 hermes-skill-health

A [Hermes Agent](https://hermes-agent.nousresearch.com/) skill for **skill-health** (devops category).

## Installation

```bash
cd ~/.hermes/skills/devops
git clone https://github.com/Marrowleaf/hermes-skill-health.git skill-health
```

## Usage

Once installed, this skill is automatically detected by Hermes Agent.

```
skill_view(name="skill-health")
```

## Structure

- `SKILL.md` — Full documentation
- `references/` — Reference materials
- `scripts/` — Utility scripts
- `templates/` — Templates
